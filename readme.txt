== Description ==

This plugin will enable a latest post block widget. If you want to use this plugin please active a theme which is developed by ashathemes.

== Installation ==

<h2>This section describes how to install the plugin and get it working.</h2>

<h4>First things first, thank you for choosing our Wp Latest Posts Block!</h4>
<h4>You can install the plugin one of two ways:</h4>
<h5 class="short-info">Install using FTP</h5>
<ul>
 	<li>Unzip the <strong>latest-posts-block.zip</strong> file locally to your machine</li>
 	<li>Connect to your FTP area using your preferred FTP package</li>
 	<li>Upload the <strong>latest-posts-block.zip</strong> folder that you extracted from the zip file to the plugin folder of your WordPress installation (<strong>wp-content -&gt; plugin</strong>)</li>
 	<li>Go to <strong>plugin </strong> and activate Easy To Top.</li>
</ul>
<h5 class="short-info">Install by WordPress (recommended)</h5>
<ul>
 	<li>Login to your WordPress admin area.</li>
 	<li>Navigate to <strong>plugin</strong> and click the <strong>Add New</strong> button at the top.</li>
 	<li>Click the <strong>Upload plugin</strong> button at the top.</li>
 	<li>Click the <strong>Choose File</strong> button and locate the <strong>latest-posts-block.zip</strong> file on your machine and click the <strong>Install Now</strong> button.</li>
 	<li>On the basis the plugin installs correctly, click the <strong>Activate</strong> link.</li>
</ul>
